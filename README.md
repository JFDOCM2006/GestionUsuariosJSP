# RegistroUsuariosWeb

Aplicación web desarrollada en Java utilizando JSP, Servlets y conexión con base de datos MySQL.

Proyecto realizado como parte de la actividad **GA7-220501096-AA2-EV02** del SENA.

## Tecnologías utilizadas

- Java 8+
- Apache Tomcat
- JSP y Servlets
- JDBC
- MySQL
- Maven
